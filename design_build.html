3 ขั้นตอนง่ายๆ

ขั้นตอนการตกแต่งภายใน
1. ลูกค้าที่มีรายละเอียด สามารถส่งข้อมูลมาทางเคาน์เตอร์ Design and Build ที่สาขา Index Living Mall หรือทาง E-mail : customer_service@indexlivingmall.com บริษัทฯ จะติดต่อกลับ พร้อมส่งใบเสนอราคาคร่าวๆ สิ่งที่ควรส่งมาด้วย แปลน บ้าน หรือ คอนโด มีขนาดประกอบด้วย
   1.1. ระบุความต้องการเบื้องต้น เช่น ต้องการตกแต่งผนังส่วนรับแขก, ชุดครัว, Walk-in closet เป็นต้น
   1.2. ระบุขนาด กว้าง x ยาว x สูง
   1.3. ระบุโทนสีที่ต้องการ เช่น ชอบสีขาวเงา (ถ้ามี)
   1.4. ระบุ Style ที่ต้องการ เช่น ต้องการตกแต่งแนว Modern, Modern Classic หรือ แนบรูปที่ชอบมาด้วยก็ได้ ส่งรูปประกอบ (ถ้ามี)
   1.5. ระบุงบประมาณคร่าวๆ เพื่อทางบริษัทฯ จะช่วยตรวจสอบว่างบประมาณสอดคล้อง กับความต้องการหรือไม่
2. ลูกค้าที่ไม่มีแปลนหรือไม่สะดวกที่จะส่งรายละเอียดมา ต้องการให้ทางบริษัทฯ เข้าไปให้ คำปรึกษาที่หน้างาน พร้อมตรวจสอบพื้นที่จริง ขั้นตอนนี้ควรโทรนัดล่วงหน้า 3-4 วัน เมื่อ เจ้าหน้าที่เข้าตรวจสอบพื้นที่ และได้รับรายละเอียดแล้ว ทางบริษัทฯจะส่งใบเสนอราคา ไม่เกิน 7 วัน ทำการ
3. ลูกค้าที่ต้องการนำแปลนเข้ามารับคำปรึกษาที่บริษัทฯ กรุณาโทรนัดล่วงหน้า 1-2 วัน ซึ่งทางบริษัทฯ จะส่งใบเสนอราคาไม่เกิน 7 วันทำการ หรือในกรณีที่รายละเอียดครบถ้วน สามารถรับการประเมินราคาได้ทันทีa
4. สำหรับลูกค้าว่าจ้างบริษัทอื่นเขียนแบบครบถ้วนแล้ว ต้องการหาผู้รับเหมาตกแต่ง ภายใน ให้โทรนัดทางบริษัทฯจะส่งเจ้าหน้าที่ไปรับแบบ ระยะเวลาในการประเมินราคา ขึ้นอยู่ กับ ปริมาณแบบ โดยปกติไม่เกิน 7-10 วัน

ขั้นตอนนี้เป็นบริการประเมินราคา
ฟรี ( สามารถส่งข้อมูลมาที่ customer_service@indexlivingmall.com )

ออกแบบด้วยโปรแกรม 3D ภาพสวยเสมือนจริง โดยค่ามัดจำในการออกแบบ "(สอบถาม เพิ่มเติม)" ขึ้นอยู่กับพื้นที่ ที่ต้องการเขียนแบบ เมื่อตัดสินใจทำสัญญา ตกแต่งภายในกับ ทางบริษัทฯจะหักค่ามัดจำแบบให้ทันที 50% ในงวดแรกที่ 2 ของสัญญา รับเหมาตกแต่ง ภายใน
สิ่งที่ลูกค้าจะได้รับ
1. แบบตกแต่งภายในเป็นภาพ 3D
2. แบบภาพรูปด้าน พร้อมบอกขนาด กว้าง x ยาว x สูง
3. แปลนระบุตำแหน่งเฟอร์นิเจอร์
4. แปลนระบุตำแหน่งจุดไฟ จุดปลั๊ก จุดทีวี จุดไฟภายในบ้าน
5. แปลนฝ้าเพดาน (ในกรณีที่มีการยกฝ้า หรือ ทำฝ้าเหลือบซ่อนไฟ)
6. แปลนพื้น
7. ประเมินราคาค่าใช้จ่ายในการตกแต่งภายใน (รู้งบประมาณคร่าวๆ ก่อนตัดสินใจ จ้างทำแบบ)
